noto che il mio input viene utilizzato nel campo src di un tag <img>, quindi decido di sfruttare onerror per eseguire un alert. 

inserisco nel campo di input x" onerror="alert(1) e noto che il sistema filtra la keyword onerror

l'ultimo tentativo è quello di inserire il payload x" oneonerrorrror="alert(1) in maniera che dopo la sostituzione di "onerror" con '' il tag risulterà <img src="x" onerror="alert(1)">
